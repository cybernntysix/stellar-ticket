# Stellar Ticket // Neural Ticketing Interface

**Stellar Ticket** is an immersive, high-fidelity ticketing platform designed for IT Support and Cybersecurity professionals. It leverages the cinematic "node constellation" aesthetic to provide a spatial, intuitive way to manage and monitor system tickets and events.

## ✧ Key Capabilities
- **Neural Constellation**: Visualize tickets as priority-colored nodes in a 3D-like space.
- **Department Drill-down**: Zoom from a high-level system view into department-specific ticket clusters.
- **Bento Dashboard**: A modular, glassmorphism command center for tracking the active queue, system logs, and knowledge base.
- **Authority Archive**: Deep-dive into ticket details with a cinematic, chronological activity log.

## 🚀 Getting Started

### Prerequisites
- Node.js (Latest LTS recommended)
- npm or yarn

### Installation
1.  **Clone the environment**:
    ```bash
    cd ~/Developer/ConstellationTicket
    npm install
    ```

2.  **Start the Neural Interface**:
    ```bash
    npm run dev
    ```
    *The system will initialize both the backend API and the Vite frontend.*
